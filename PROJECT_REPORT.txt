------------------------------------------------------------
1. Overview
------------------------------------------------------------
ResumeSense is a full-stack, cross-platform application for parsing, analyzing, and visualizing resume data. It leverages advanced AI/NLP libraries to extract structured information from resumes in various formats (PDF, DOCX, images, etc.), provides analytics dashboards, and supports both single and bulk resume processing. The platform features a modern, futuristic UI with dark/light mode and export options.

------------------------------------------------------------
2. Tech Stack
------------------------------------------------------------
- **Backend:** FastAPI (Python)
- **Frontend:** React (JavaScript), TailwindCSS
- **Data Science/NLP:** SpaCy, Presidio, Matplotlib, Pandas, Cohere (optional)
- **Visualization:** Chart.js
- **Others:** PyPDF2, python-docx, Pillow, pytesseract, opencv-python

------------------------------------------------------------
3. Key Features
------------------------------------------------------------
- Multi-format resume parsing (PDF, DOCX, TXT, images)
- AI-powered extraction (emails, phones, skills, entities, etc.)
- Bulk upload and batch processing
- Analytics dashboard with charts and summary
- Anonymization (PII removal)
- Export results as CSV or JSON
- Responsive, modern UI with dark/light mode
- Animated, glowing footer: "Engineered by Abhishek"

5. Backend Details
------------------------------------------------------------
- **main.py**: FastAPI app, routes for parsing, analytics, suggestions, etc.
- **bulk.py**: Handles multiple file uploads, processes each resume, aggregates results.
- **analytics.py**: Generates analytics (summary, charts) using Pandas and Matplotlib. Uses 'Agg' backend for server-side rendering.
- **extractor.py**: Extracts text and entities from files using SpaCy, PyPDF2, python-docx, Pillow, pytesseract.
- **nlp.py**: NLP utilities for skill extraction, entity recognition, etc.
- **match.py**: Compares resume with job description using NLP and semantic similarity.
- **suggest.py**: Integrates with Cohere API for AI-powered suggestions and scoring.
- **requirements.txt**: All dependencies listed for cross-platform compatibility.

------------------------------------------------------------
6. Frontend Details

8. How It Works (Functioning)
------------------------------------------------------------
1. **User uploads resume(s)** via the frontend (single or bulk).
2. **Frontend** sends files to backend via REST API.
3. **Backend** extracts text and entities using NLP and AI libraries.
4. **Analytics**: Backend computes summary stats and generates charts (Matplotlib, Chart.js).
5. **Results** are returned to the frontend and displayed in an animated dashboard.
6. **User can export** results as CSV/JSON. Optionally, anonymize sensitive info.
7. **Footer** always shows "Engineered by Abhishek" in a glowing, futuristic style.
